## 🚀 Getting Started

1. **Clone the repo**

   ```bash
   git clone https://github.com/Draftgamz/portfolio-v-6.git
   cd portfolio-v-6
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the dev server**

   ```bash
   npm run dev
   ```

4. **Build for production**

   ```bash
   npm run build
   ```

5. **Preview the production build**

   ```bash
   npm run preview
   ```
